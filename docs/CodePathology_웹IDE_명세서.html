프로젝트 초기화
인프라
CodePathology 프로젝트를 초기화해줘. VS Code Extension 없이 웹 앱으로만 구성. 구조: codepathology/ ├── backend/ (Python FastAPI) ├── frontend/ (React + TypeScript + Vite) ├── docker-compose.yml (PostgreSQL 16 + Redis 7 + backend + frontend) backend/requirements.txt: fastapi, uvicorn[standard], sqlalchemy[asyncio], asyncpg, alembic, pydantic[email], python-jose[cryptography], passlib[bcrypt], httpx, openai, langchain, langchain-openai, faiss-cpu, tree-sitter, py-tree-sitter-languages, redis, python-dotenv, websockets frontend/package.json: react 18, typescript, vite, react-router-dom 6, @monaco-editor/react, recharts, d3, axios, zustand, tailwindcss, @headlessui/react docker-compose.yml: - postgres:16 (port 5432, POSTGRES_DB=codepathology) - redis:7 (port 6379) - backend (port 8000, depends_on postgres+redis) - frontend (port 3000) .env.example: DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/codepathology REDIS_URL=redis://localhost:6379 OPENAI_API_KEY=sk-xxx JWT_SECRET=your-secret-key CORS_ORIGINS=http://localhost:3000

AI
Tree-sitter 코드 분석 서비스. services/ast_parser.py: - parse_code(code: str, language: str) → dict - Java/Python/JS 지원 (py-tree-sitter-languages 사용) - 반환: {functions, classes, loops, variables, error_prone_patterns, complexity_score} - detect_patterns(ast_result: dict) → list[dict] - off-by-one, 무한루프 위험, null 체크 누락, 미사용 변수 감지 - 각 패턴: {pattern_name, line, description, severity} 파싱 실패 시 graceful fallback (빈 AST + raw code 반환).

AI 병리 진단 엔진
AI
GPT-4o 병리 진단 서비스. services/ai_diagnosis.py: - async diagnose_code(code, language, ast_result, student_history, curriculum_context) → DiagnosisResult 시스템 프롬프트: """당신은 "코드 병리학자(Code Pathologist)"입니다. 수강생이 제출한 코드를 분석하되, "어디가 틀렸는지"가 아니라 "이 수강생의 사고 패턴에 어떤 구조적 결함이 있는지"를 진단하세요. JSON 형식으로 반환: { "pathology_name": "한국어 병리명", "pathology_code": "ENGLISH_CODE", "category": "관련 개념", "severity": "low|medium|high|critical", "symptom": "코드에서 관찰되는 증상 (줄 번호 포함)", "root_cause": "근본적 사고 결함", "prescription": "처방 (마이크로 실습 2~3개 포함)", "meta_insight": "수강생이 자기 사고 패턴을 인식할 메타인지 메시지" } 병리 없으면 {"pathology_name":"정상","severity":"none"}""" - check_recurrence(student_id, pathology_code) → (bool, int) - OpenAI: model="gpt-4o", temperature=0.3, response_format=json_object, timeout=30s, 3회 재시도

✅ 산출물

소크라테스식 힌트 생성기
AI
